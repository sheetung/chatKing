# chatKing

一个用于生成聊天排名排行榜图片的 LangBot 插件。

## 功能

- 根据聊天活动生成排名排行榜
- 支持自定义时间范围（例如：1天、2天排行榜）
- 通过 API 请求创建可视化排名图片
- 通过插件设置轻松配置

## 安装

通过LangBot插件市场安装

## 使用

### 命令格式

要生成排名排行榜，请发送以下格式的消息：

```
[数字]日发言榜
```

示例：
- `1日发言榜` - 生成当天的排行榜
- `7日发言榜` - 生成过去7天的排行榜

### 工作原理

1. 插件监控群消息并记录聊天活动
2. 当收到排名命令时，查询数据库获取聊天统计数据
3. 向配置的 API 发送请求以生成排名图片
4. 生成的图片被发送回群组
5. 如果图片生成失败，则发送文本形式的排行榜

## 配置

插件可以通过以下设置进行配置：

| 设置 | 描述 | 默认值 |
|------|------|--------|
| `api_url` | 生成排名图片的 API 端点 | `contact author for free` |
| `access_token` | API 的访问令牌 | `contact author for free` |

## 依赖

- `requests` - 用于发起 API 请求
- `langbot-plugin` - 基础插件框架
